# SAHC Homepage Redesign Plan

## Branch
`feat/sahc-homepage-redesign`

## Goal
Improve the homepage by fixing information hierarchy and making the site's primary utility easier to discover.

## Problem
The current homepage introduces the organization background before showing visitors the site's main utility. The Directory appears too far down the page.

## Decisions
1. Keep the existing hero/title area.
2. Add a short purpose paragraph near the top of the homepage.
3. Move the Directory section upward so it appears before the About/Background section.
4. Keep the About SAHC / Background content, but move it below the Directory.
5. Keep membership-related information lower on the page.

## Target Content Order
1. Hero / Site Title
2. Purpose of the Site
3. Directory
4. About SAHC / Background
5. Who Can Join?
6. Membership Cost
7. Benefits of Membership
8. Footer / related links / contact

## Purpose Paragraph Draft
This website serves as a shared hub for the Sacramento Area History Consortium, helping visitors discover local history organizations, explore regional resources, follow events, and connect with the network of groups working to preserve and promote Sacramento-area history.

## Design Intent
- Prioritize clarity over redesign complexity.
- Emphasize function early by surfacing the Directory near the top.
- Reduce the sense that the page is primarily administrative.
- Preserve existing content where possible, mostly by reordering rather than rewriting.

## Implementation Notes
- Reuse current homepage content.
- Reorder sections instead of doing a full content rewrite.
- Add the new purpose paragraph directly below the hero/title.
- Keep visual changes light unless needed to support the new hierarchy.
- Optional later enhancement: overlay site title/tagline on the hero image.
